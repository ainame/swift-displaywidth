# Changelog

All notable changes to this project will be documented in this file.

## [0.1.0] - 2026-04-06

### Added

- Added ANSI escape sequence stripping during string display width measurement.
- Added tab-aware string measurement with support for tab stops and fixed spaces.
- Added test coverage for embedded terminal image escapes.

### Changed

- Improved README usage documentation for ANSI-aware and tab-aware measurement.
- Optimized processed string measurement performance.

### Fixed

- Addressed follow-up review feedback in the ANSI and tab processing implementation.
